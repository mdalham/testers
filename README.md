🚀 Testers Community App

A powerful Flutter-based platform where developers and testers collaborate to improve apps before public release.
Publish apps, join testing groups, track progress, and provide valuable feedback — all in one place.

📱 Features
👨‍💻 For Developers (Publishers)
Create and manage testing groups
Add app details with Play Store links
Set required testing duration
Approve or reject testers
Track tester activity

🧑‍🔬 For Testers
Join testing groups بسهولة
Discover new apps before launch
Submit feedback & reviews
Track remaining testing days
Earn rewards/coins (optional feature)

🔔 Notification System (In-App)
New user joined group
App testing started
Testing completed
Group updates & reminders

🎨 UI Highlights
Clean & modern Material 3 design
Dark & Light mode support
Smooth animations
Responsive layout

🧱 Tech Stack
Flutter (UI Framework)
Dart
Firebase Authentication
Cloud Firestore (Database)
Hive / Local Storage

Core Functionalities
User authentication (Login/Register)
Role-based system (Tester / Publisher)
Group creation & joining system
App listing & testing workflow
Approval system for testers
Remaining days tracker
